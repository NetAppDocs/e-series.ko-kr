= 호스트 포트 프로토콜 변환
:icons: font
:imagesdir: ../media/


[role="lead"]
호환성을 위해 호스트의 프로토콜을 다른 프로토콜로 변환할 수 있으며, 호스트 연결의 프로토콜을 인프라 요구에 맞게 다른 프로토콜로 변환할 수 있습니다.
