= 하드웨어 설치 - EF50 및 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50 또는 EF80 스토리지 시스템 설치를 준비한 후 캐비닛 또는 통신 랙에 랙 마운트 하드웨어를 설치한 다음 스토리지 시스템을 설치하고 고정합니다.

스토리지 시스템이 캐비닛과 함께 제공된 경우 이 절차를 건너뛰고 link:install-cable.html["하드웨어에 케이블 연결"]로 이동할 수 있습니다.

.시작하기 전에
* 평평하고 정전기가 없는 작업 영역을 준비합니다.
* ESD 밴드를 착용하고 정전기 방지 조치를 취하십시오.
* 랙 마운트 하드웨어와 함께 포장된 설치 지침이 있는지 확인하십시오. 없는 경우 link:../rackmount-hardware.html["랙 마운트 하드웨어"^]을(를) 참조하십시오.
* 스토리지 시스템의 무게와 관련된 리프팅 주의 사항을 숙지하십시오. link:install-prepare.html#step-2-understand-the-lifting-precautions["리프팅 주의 사항 이해"^]을 참조하십시오.
* 스토리지 시스템을 통한 공기 흐름이 드라이브가 설치된 전면에서 유입되어 포트가 있는 후면으로 배출된다는 점을 이해하십시오.


.단계
. 랙 마운팅 하드웨어에 포함된 지침을 사용하여 스토리지 시스템용 랙 마운팅 하드웨어를 설치합니다.
+

CAUTION: 캐비닛 또는 랙에서 사용 가능한 가장 낮은 공간에 랙 마운팅 하드웨어를 설치하여 스토리지 시스템 설치 시 넘어지는 것을 방지하십시오.

. 캐비닛 또는 통신 랙에 스토리지 시스템을 설치하고 고정하십시오.
+
.. 스토리지 시스템을 레일 위에 놓고 스토리지 시스템 아래쪽에서 받쳐 제자리에 밀어 넣으십시오.
.. 동봉된 장착 나사를 사용하여 스토리지 시스템을 캐비닛 또는 통신 랙에 고정하십시오.


. 스토리지 시스템 전면에 베젤을 부착합니다.
+
.. 베젤을 스토리지 시스템 앞에 배치하여 양쪽 끝의 구멍이 스토리지 시스템의 고정 장치와 정렬되도록 합니다.
.. 베젤을 제자리에 끼웁니다.




.다음 단계
