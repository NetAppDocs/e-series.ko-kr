= 하드웨어 전원 켜기 - EF50 및 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50 또는 EF80 스토리지 시스템에 대한 컨트롤러 간 미러링 및 호스트 연결을 케이블로 연결한 후 컨트롤러의 전원을 켭니다.

.시작하기 전에
정전기 방지 예방 조치를 취하십시오.

.단계
. 각 컨트롤러 전원 공급 장치에 전원 케이블을 연결한 다음, 서로 다른 회로의 전원에 연결하십시오.
+

NOTE: 타사 PDU(전원 분배 장치)를 사용하는 경우 컨트롤러 바로 뒤에 있는 전원 콘센트를 사용하지 마십시오. 컨트롤러에 대한 액세스가 차단될 수 있습니다.

+
* 전원 케이블 *

+
image:../media/power_cable_inst-hw-ef600.png["전원 케이블"]

+
image::../media/drw_psu_layout_1_ieops-1886.svg[스토리지 시스템 전원 연결 다이어그램]

+
** 시스템 부팅이 시작됩니다. 초기 부팅에는 최대 8분이 소요될 수 있습니다.
** 시스템 섀시의 LED가 깜빡이고 팬이 작동하기 시작하면 컨트롤러에 전원이 공급되고 있음을 나타냅니다.
+
시동 초기에는 팬 소음이 매우 큰 것이 정상입니다.

** 시스템 섀시 전면(베젤 뒤)의 디지털 디스플레이에 불이 들어오지 않습니다.


. 각 전원 공급 장치의 고정 장치를 사용하여 전원 코드를 고정합니다.


.다음 단계
스토리지 시스템의 전원을 켠 후 link:install-complete-setup.html["전체 스토리지 시스템 설정"].
