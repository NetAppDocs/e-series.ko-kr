.이 작업에 대해
컨트롤러 하나로 시작한 다음 다른 컨트롤러에 대한 단계를 반복합니다.

.단계
. 아직 접지되지 않은 경우 올바르게 접지하십시오.
. 컨트롤러 중 하나에서 제거하려는 I/O 모듈에서 케이블을 분리합니다. 단, I/O 블랭킹 모듈인 경우는 제외합니다.
+
케이블에 라벨을 붙여 어디에서 왔는지 알 수 있도록 할 수 있습니다. 하지만 업그레이드하려는 I/O 모듈의 종류에 따라 동일한 포트에 다시 케이블을 연결하지 않거나 기존 케이블을 재사용할 수 없을 수도 있습니다.

. 대상 슬롯에서 I/O 모듈 또는 I/O 블랭킹 모듈을 제거합니다.
+
다음 그림은 I/O 블랭킹 모듈을 제거하는 모습을 보여줍니다.

+
image::../media/drw_g_io_blanking_module_replace_ieops-1901.svg[I/O 블랭킹 모듈 제거]

+
[cols="1,4"]
|===


 a| 
image:../media/icon_round_1.png["설명선 번호 1"]
 a| 
I/O 모듈 또는 I/O 블랭킹 모듈에서 나비 나사를 시계 반대 방향으로 돌려 풉니다.



 a| 
image:../media/icon_round_2.png["설명선 번호 2"]
 a| 
왼쪽의 탭과 나사를 사용하여 컨트롤러에서 I/O 모듈 또는 I/O 블랭킹 모듈을 빼냅니다.

|===
. 새 I/O 모듈의 포장을 풀고 스토리지 시스템 근처의 평평하고 정전기가 없는 표면에 놓습니다.
. 새 I/O 모듈을 설치합니다.
+
.. I/O 모듈을 컨트롤러 슬롯 개구부의 가장자리에 맞춰 정렬합니다.
.. I/O 모듈을 슬롯에 완전히 밀어 넣고 모듈이 커넥터에 제대로 장착되었는지 확인하십시오.
+
왼쪽 탭과 엄지 나사를 사용하여 I/O 모듈을 밀어 넣을 수 있습니다.

.. 엄지나사를 시계 방향으로 돌려 조입니다.


. I/O 모듈을 호스트 네트워크에 케이블로 연결하십시오.
+
호스트 네트워크 케이블 연결에는 시스템 설치 및 설정 콘텐츠의 link:../install-hw-ef50-ef80/install-cable.html["하드웨어에 케이블 연결"^] 섹션을 사용할 수 있습니다.

. 다른 컨트롤러의 I/O 모듈을 업그레이드하려면 이 단계를 반복하십시오.
