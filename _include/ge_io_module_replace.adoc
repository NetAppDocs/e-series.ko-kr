장애가 발생한 I/O 모듈을 교체하려면 컨트롤러에서 해당 모듈을 찾아 특정 단계 순서를 따르십시오.

.단계
. 아직 접지되지 않은 경우 올바르게 접지하십시오.
. 장애가 발생한 I/O 모듈에서 케이블을 분리합니다.
+
케이블이 어디에서 왔는지 알 수 있도록 라벨을 붙여 두십시오.

. 컨트롤러에서 장애가 발생한 I/O 모듈을 제거합니다.
+
image::../media/drw_g_io_module_replace_ieops-1900.svg[I/O 모듈 제거]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[설명선 번호 1]
 a| 
I/O 모듈 나사를 시계 반대 방향으로 돌려 풀어줍니다.



 a| 
image::../media/icon_round_2.png[설명선 번호 2]
 a| 
왼쪽에 있는 포트 라벨 탭과 나사를 사용하여 I/O 모듈을 컨트롤러에서 빼냅니다.

|===
. 교체용 I/O 모듈을 대상 슬롯에 설치합니다.
+
.. I/O 모듈을 슬롯 가장자리에 맞춰 정렬합니다.
.. I/O 모듈을 슬롯에 완전히 밀어 넣고 모듈이 커넥터에 제대로 장착되었는지 확인하십시오.
+
왼쪽 탭과 엄지 나사를 사용하여 I/O 모듈을 밀어 넣을 수 있습니다.

.. 엄지나사를 시계 방향으로 돌려 조입니다.


. I/O 모듈에 케이블을 연결합니다.
