.단계
. 스토리지 시스템의 호스트 I/O 모듈에 있는 프로토콜을 모르는 경우 SANtricity System Manager를 사용하여 확인하십시오.
+
.. *설정* > *시스템*을 선택합니다.
.. *추가 설정* 섹션에서 *호스트 I/O 프로토콜 변경*을 선택하여 해당 페이지를 엽니다.
.. *호스트 I/O 프로토콜* 필드에 표시되는 기본 프로토콜을 참조하십시오.
+

NOTE: 호스트 I/O 모듈에서 사용 가능한 프로토콜 목록을 보려면 *호스트 I/O 프로토콜* 필드의 드롭다운 메뉴를 선택하십시오. 프로토콜 목록은 현재 설치된 호스트 I/O 모듈을 기반으로 합니다.



. 다음 단계는 기본 프로토콜이 원하는 프로토콜인지 여부에 따라 달라집니다.
+
[cols="1,2"]
|===
| 만약... | 그런 다음... 


 a| 
기본 프로토콜은 원하는 프로토콜입니다
 a| 
별도의 조치가 필요하지 않습니다. 이 절차를 종료하셔도 됩니다.



 a| 
기본 프로토콜이 원하는 프로토콜이 아닙니다
 a| 
이 절차를 계속 진행하여 프로토콜을 변경하십시오.

|===
. 이 절차를 위해 다운타임 유지보수 윈도우를 예약하십시오.
