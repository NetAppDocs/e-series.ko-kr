.단계
. 아직 접지되지 않은 경우 올바르게 접지하십시오.
. NV 배터리를 찾습니다.
. NV 배터리를 제거합니다.
+
image::../media/drw_g_nv_battery_replace_ieops-1864.svg[NV 배터리 교체]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[설명선 번호 1]
 a| 
NV 배터리를 들어 올려 수납함에서 꺼냅니다.



 a| 
image::../media/icon_round_2.png[설명선 번호 2]
 a| 
배선 하네스를 고정 장치에서 분리하십시오.



 a| 
image::../media/icon_round_3.png[설명선 번호 3]
 a| 
.. 커넥터의 탭을 안쪽으로 밀어 넣고 고정합니다.
.. 커넥터를 소켓에서 위쪽으로 당겨 빼냅니다.
+
위로 당기면서 커넥터를 끝에서 끝까지(길이 방향으로) 부드럽게 흔들어 분리하십시오.



|===
. 교체용 NV 배터리의 포장을 풀고 스토리지 시스템 근처의 평평하고 정전기가 없는 표면에 놓습니다.
+
결함이 있는 NV 배터리를 반환할 때 사용할 포장재를 보관하십시오.

. 교체용 NV 배터리를 설치합니다.
+
.. 배선 커넥터를 소켓에 꽂습니다.
.. 전원 공급 장치 측면을 따라 배선을 연결하고, 고정 장치에 넣은 다음, NV 배터리 칸 앞쪽의 채널을 통해 빼냅니다.
.. NV 배터리를 해당 칸에 넣으십시오.
+
NV 배터리는 해당 칸에 평평하게 장착되어야 합니다.
