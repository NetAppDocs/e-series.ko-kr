= 스토리지 배열의 이름을 바꿉니다
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
vCenter용 Storage Plugin의 관리 페이지에 표시되는 스토리지 배열의 이름을 변경할 수 있습니다.

.단계
. Manage * 페이지에서 스토리지 배열 이름 왼쪽에 있는 확인란을 선택합니다.
. 행의 맨 오른쪽에 있는 줄임표를 선택한 다음 팝업 메뉴에서 * 스토리지 배열 이름 바꾸기 * 를 선택합니다.
. 새 이름을 입력하고 * 저장 * 을 클릭합니다.
