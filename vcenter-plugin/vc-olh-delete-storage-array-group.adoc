= 스토리지 어레이 그룹을 삭제합니다
:hardbreaks:
:allow-uri-read: 
:experimental: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
더 이상 필요하지 않은 스토리지 그룹을 하나 이상 제거할 수 있습니다.

이 작업은 스토리지 어레이 그룹만 삭제합니다. 삭제된 그룹과 연결된 스토리지 배열은 모두 관리 보기 또는 연결된 다른 그룹을 통해 액세스할 수 있습니다.

.단계
. 관리 페이지에서 메뉴 관리 그룹 [스토리지 어레이 그룹 삭제]를 선택합니다.
. 삭제할 스토리지 그룹을 하나 이상 선택합니다.
. 삭제 * 를 클릭합니다.
