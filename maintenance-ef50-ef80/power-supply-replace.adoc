= 전원 공급 장치 교체 - EF50 및 EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50 또는 EF80 스토리지 시스템의 전원 공급 장치(PSU)가 고장 나거나 오작동하는 경우 교체하여 시스템이 안정적으로 작동하는 데 필요한 전력을 지속적으로 공급받도록 하십시오.

.시작하기 전에
* Recovery Guru에서 세부 정보를 검토하여 전원 공급 장치에 문제가 있는지 확인하십시오.
* 전원 공급 장치의 주의(황색) LED가 켜져 있는지 확인하십시오. 이는 전원 공급 장치 또는 내장 팬에 결함이 있음을 나타냅니다.
* 다른 스토리지 시스템 문제를 해결하십시오.
+
Recovery Guru에서 *다시 확인*을 선택하여 다른 문제가 있는지 확인하십시오.

* 다음 사항을 확인하십시오.
+
** 교체 전원 공급 장치.
+

NOTE: 전압 유형이 다른 PSU를 혼용하지 마십시오. 항상 Like를 사용합니다.

** ESD 손목 밴드를 착용했거나 다른 정전기 방지 조치를 취했습니다.
** 정전기가 없는 평평한 작업 표면
** 장애가 발생한 컨트롤러의 SANtricity System Manager에 액세스할 수 있는 브라우저가 있는 관리 스테이션.
+
System Manager 인터페이스를 열려면 브라우저에서 장애가 발생한 컨트롤러의 도메인 이름 또는 IP 주소를 가리킵니다.





.이 작업에 대해
* 전원 공급 장치에 장애가 발생하면 스토리지 시스템에 이중화된 전원이 확보되도록 가능한 한 빨리 교체해야 합니다.
* 전원 코드를 결함이 있는 PSU에서 뽑고 교체한 다음 전원 코드를 다시 연결하여 전원 공급 장치를 교체하십시오.
* 스토리지 시스템 위치(파란색) LED(스토리지 시스템 전면과 두 컨트롤러에 있음)를 켜서 영향을 받는 스토리지 시스템을 물리적으로 찾을 수 있습니다. SANtricity System Manager를 사용하여 *하드웨어* > *컨트롤러 및 구성 요소*를 선택하고 *컨트롤러 셸프* 탭을 선택한 다음 상황 메뉴에서 *위치 표시등 켜기*를 선택하십시오.


.단계
. 아직 접지되지 않은 경우 올바르게 접지하십시오.
. 전원 코드 고정 장치를 열어 전원 공급 장치에서 전원 코드를 분리한 다음 전원 공급 장치에서 전원 코드를 뽑으십시오.
+

NOTE: 전원 공급 장치에는 전원 스위치가 없습니다.

. 전원 공급 장치를 제거하십시오.
+
image::../media/ef50-ef80_psu_ac_orange_tab_replace_ieops-2611.svg[AC 전원 공급 장치 교체]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[설명선 번호 1]
 a| 
전원 공급 장치 손잡이를 위쪽으로 돌려 수평 위치로 만든 다음 잡으세요.



 a| 
image::../media/icon_round_2.png[설명선 번호 2]
 a| 
엄지손가락으로 주황색 잠금 탭을 눌러 전원 공급 장치를 컨트롤러에서 분리합니다.



 a| 
image::../media/icon_round_3.png[설명선 번호 3]
 a| 
한 손으로 전원 공급 장치의 무게를 받쳐준 상태에서 다른 손으로 컨트롤러에서 전원 공급 장치를 빼내십시오.


CAUTION: PSU의 길이가 짧습니다. 컨트롤러에서 분리할 때 갑자기 컨트롤러에서 흔들려 부상을 입지 않도록 항상 양손으로 받쳐 주십시오.

|===
. 교체용 전원 공급 장치의 포장을 풀고 스토리지 시스템 근처의 평평한 표면에 놓습니다.
+
고장난 전원 공급 장치를 반환할 때 사용할 수 있도록 포장재를 모두 보관하십시오.

. 교체용 전원 공급 장치를 설치합니다.
+
.. 양손을 사용하여 전원 공급 장치의 가장자리를 컨트롤러의 구멍에 맞춰 받쳐줍니다.
.. 주황색 잠금 탭이 딸깍 소리를 내며 제자리에 고정될 때까지 전원 공급 장치를 컨트롤러에 부드럽게 밀어 넣으십시오.
+
PSU는 내부 커넥터와 한 방향으로만 제대로 연결되고 고정됩니다.

+

NOTE: 내부 커넥터가 손상되는 것을 방지하기 위해 전원 공급 장치를 컨트롤러에 끼울 때 과도한 힘을 가하지 마십시오.

.. 손잡이를 아래쪽으로 돌려서 정상 작동에 방해가 되지 않도록 하십시오.


. 전원 코드를 전원 공급 장치에 다시 연결하고 전원 코드 고정 장치를 사용하여 전원 코드를 고정하십시오.
+
전원 공급 장치에 전원이 복구되면 상태 LED가 녹색으로 표시되어야 합니다.

. 전원 공급 장치의 상태가 최적인지, 그리고 스토리지 시스템의 주의 LED가 꺼져 있는지 확인하십시오.
+
상태가 최적이 아니거나 주의 LED가 켜져 있는 경우 전원 케이블이 올바르게 장착되어 있고 전원 공급 장치가 올바르게 설치되어 있는지 확인하십시오. 필요한 경우 전원 공급 장치를 분리했다가 다시 설치하십시오.

+

NOTE: 문제를 해결할 수 없는 경우, 이 절차를 계속 진행하기 전에  https://mysupport.netapp.com/site/global/dashboard["NetApp 지원"]에 문의하십시오.

. SANtricity System Manager에서 *Support > Upgrade Center*를 클릭하여 최신 버전의 SANtricity OS가 설치되어 있는지 확인하십시오.
+
필요한 경우 최신 버전을 설치합니다.

. 키트와 함께 제공된 RMA 지침에 설명된 대로 오류가 발생한 부품을 NetApp에 반환합니다.
+
RMA 번호가 필요하거나 교체 절차에 대한 추가 지원이 필요한 경우 https://mysupport.netapp.com/site/global/dashboard["NetApp 지원"], 888-463-8277(북미), 00-800-44-638277(유럽) 또는 +800-800-80-800(아시아/태평양)으로 기술 지원 팀에 문의하십시오.//2025-11-17 ontap-systems-internal/issues/1391
