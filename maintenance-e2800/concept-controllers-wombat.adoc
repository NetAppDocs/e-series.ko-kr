= 컨트롤러
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
컨트롤러는 보드, 펌웨어 및 소프트웨어로 구성됩니다. 드라이브를 제어하고 System Manager 기능을 구현합니다. 이 섹션에서는 두 번째 컨트롤러를 추가하여 이중 구성으로 변환하는 방법과 단면 인쇄 구성 및 양면 인쇄 구성에서 컨트롤러를 교체하는 방법에 대해 설명합니다.
